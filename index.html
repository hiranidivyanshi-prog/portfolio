Project Case Studies

01
Academic Performance Dashboard
Tableau · Looker Studio · ERP Data · 5,000+ Records
The Problem
Senior leadership at CMS had no real-time visibility into student attendance and academic performance. Reports were produced manually in Excel, taking hours each week and arriving too late to act on.
My Approach
Extracted and cleaned 5,000+ ERP records using SQL. Built automated data pipelines to feed live dashboards in Tableau and Looker Studio with drill-down by class, subject, and teacher.
Key Findings & Impact
📉 Students with attendance below 75% were 3× more likely to underperform — flagged for early intervention.
⚡ Automated reporting reduced manual effort by 40%, freeing ~6 hours/week per reporting cycle.
📊 8+ KPI dashboards deployed and actively used by leadership for real-time monitoring.
✅ Data accuracy improved to 100% through validation rules built into the pipeline.

02
Student Performance & Attendance Analysis
Python · SQL · Pandas · Matplotlib · Seaborn
The Problem
Academic planners needed to understand which factors most strongly predicted student performance drops, so resources could be allocated to at-risk groups proactively.
My Approach
Conducted end-to-end EDA using Python (Pandas, Seaborn, Matplotlib). Used SQL to aggregate student records. Studied correlations between attendance, subject scores, and performance outcomes.
Key Findings & Impact
📌 Strong positive correlation (r = 0.76) found between attendance rate and final exam scores.
📌 Math and Science showed highest performance variance — targeted for additional support sessions.
📌 Identified a critical attendance drop-off in months 3–4 of each semester — early alert system recommended.
📌 Findings compiled into a structured report used by the academic committee for the next planning cycle.
